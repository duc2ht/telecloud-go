# 📱 Mobile Client Guide / Hướng dẫn Client di động

This guide explains how to connect and use the mobile client app for TeleCloud.
Hướng dẫn này giải thích cách kết nối và sử dụng ứng dụng client di động cho TeleCloud.

---

## 🇻🇳 Tiếng Việt

### 1. Giới thiệu Client di động

TeleCloud cung cấp phiên bản client di động tối ưu, bạn có thể [tải và sử dụng tại đây](https://telecloud-app.dabeecao.org).
Ứng dụng được thiết kế tối ưu hóa cho màn hình cảm ứng điện thoại di động, đem lại trải nghiệm mượt mà, hỗ trợ phát nhạc/video chạy nền, đọc sách EPUB/PDF, truyện tranh CBZ và đồng bộ hóa ngoại tuyến.


### 2. Cách kết nối với Server của bạn

Để kết nối Client di động với máy chủ TeleCloud tự chạy (self-hosted) của bạn:

1. **Chuẩn bị tên miền Server**: Đảm bảo máy chủ TeleCloud của bạn được thiết lập tên miền riêng hỗ trợ kết nối bảo mật HTTPS (ví dụ: `https://cloud.tenmien-cua-ban.com`).
2. **Nhập URL máy chủ**: Khi mở ứng dụng di động lần đầu, bạn sẽ thấy ô nhập **Server URL**. Hãy điền đầy đủ địa chỉ HTTPS của server bạn vào.
3. **Đăng nhập**: Sử dụng tài khoản đăng nhập của bạn (Admin hoặc tài khoản con) đã được tạo trên hệ thống.
4. **Trải nghiệm**: Hệ thống tự động ghi nhớ phiên đăng nhập và kết nối của bạn cho những lần sử dụng sau.

## 🇺🇸 English

### 1. Introduction to Mobile Client

TeleCloud provides an optimized mobile client, which you can [access and download here](https://telecloud-app.dabeecao.org).
The app is tailored for mobile touch screens, offering smooth navigation, background media playing, EPUB/PDF reading, CBZ comic viewing, and offline synchronization.


### 2. How to Connect to Your Server

To connect the Mobile Client to your self-hosted TeleCloud server:

1. **Server Domain Requirement**: Ensure your self-hosted TeleCloud server is configured with a custom domain and supports secure HTTPS (e.g., `https://cloud.your-domain.com`).
2. **Enter Server URL**: When opening the mobile client for the first time, you will be prompted with a **Server URL** field. Enter your full HTTPS server address.
3. **Log in**: Enter your credentials (Admin or child account) created on your server.
4. **Enjoy**: The app will automatically remember your server configuration and login session for future access.
